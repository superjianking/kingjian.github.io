那我只能用字典来跑密码了哦，这种小功能，肯定只有交给 python 来完成了，搜了一下发现 pywifi 这个组件刚好满足需求。

# 思路

对给定的 wifi 名称，我们循环使用密码去一一做尝试，如果失败就断开连接，直到连接成功为止。
